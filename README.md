## Sobre 📄
O MovieMatch é um sistema de recomendação de filmes que cria sugestões personalizadas combinando as melhores características de dois filmes distintos. Perfeito para você que está procurando um filme que combine suspense e fantasia. Ou para você que adora filmes de suspense, mas seu amigo prefere as comédias.
